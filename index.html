OBJETIVOS

El presente Objeto virtual de información, pretende establecer los principales conceptos necesarios para el manejo adecuado del sql server, con el fin de estar en la capacidad de realizar distintos llamados a la base de datos mediante consultas "Querys" de tal manera que el estudiante cuente con lo básico para a su manejo y comprensión de tal manera que se note la importancia de su uso para cualquier desarrollador o programador.

Utilizar elementos didácticos que aumenten el interés y la capacidad de compresión acerca de cómo utilizar las proncipales sentencias de SQL SERVER, sus palabras clave, funciones, entre otras.

Crear un espacio que sirva como guía inicial y pueda ser utilizada para adquirir los conocimientos básicos de la creacion de un query basico, el manejo de vistas, y funciones principales para efectuar el CRUD, tales como, INSERT INTO, DELETE, UPDATE, SELECT. Me diante el acceso a las lecturas, videos y actividades evaluativas cortas, que aportan mucho valor al proceso de aprendizaje.
